# Локальная приёмка release candidate

Дата: 2026-07-12

## Решение

Локальный release candidate принят для внутреннего плейтеста. Серверная публикация и продуктовый релиз не входят в это решение.

## Обязательные проверки

- `npm test` — 121/121.
- `npm run balance` — `Release balance gate: ok=true`.
- `npm run smoke` — 2/2, включая полную детерминированную победу.
- Desktop browser smoke — загрузка HUD, игрового поля, лабиринта, настроек эффектов и диагностики save без ошибок консоли.
- Mobile browser smoke — компоновка `390×844` загружается без горизонтального переполнения основных панелей и без ошибок консоли.
- Валидный импорт save переживает reload; ошибочный импорт сохраняет текущий прогресс.

## Известные риски

- Внешний плейтест ещё не проведён; GAME-027 сознательно пропущена без выдумывания наблюдений.
- Acquisition budget остаётся advisory-диагностикой: короткий финальный gate является текущим combat-paced решением.
- При очень низком FPS активная вкладка может считать игровое время медленнее реального; скрытая вкладка использует отдельный offline catch-up.
- Финальные музыка, арт и расширенный набор звуков не входят в текущий MVP.

## Следующий backlog

Эти задачи не входят в текущий release candidate:

1. Провести 3-5 внешних плейтестов по `docs/PLAYTEST.md` и только после этого сформировать не более пяти проверяемых изменений.
2. Добавить автоматизированный browser smoke для desktop и mobile viewport в CI.
3. Решить по данным плейтеста, сохранять ли короткий L5 gate или менять экономику.
4. Отдельно спроектировать серверную синхронизацию save и аккаунты.
